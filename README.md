<h1 align="center">
  BEZIER CURVE
</h1>

<h2 align="center">Javascript implementation to visualize bezier curves</h2>

## 🔧 Resources
* Move in canvas.
* Create bezier curves.
* Delete created curve.
* Plot curvature animation.
* Plot derivative animation.

## ❓ How to use?

### Run

```bash
$ cd BEZIER-CURVE
$ browsername ./index.html
```
